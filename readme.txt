Plug-In Description:

The responsive Goolge Maps Plug-In brings Responsive Google Maps to your site.
When the browser-window is resized, the Plug-In automatically recenters the map view to the desired coordinates. You can activate and deactivate map control elements, set the height and the type of map (road, satellite).

And that’s pretty much it.

Installation:

Just download and install the Plug-In as usual and you are good to go.

Minimalistic example:

[responsive_map lat="51.44303" lng="7.01247"]

Possible Parameters:

lng (float): 51.44303  / Longitude
lat (float): 7.01247 / Latitude
style (string) = null / Display normal map Styling or set it to greyscale by setting style to „monochrome“
height (str) = 400px / Height of the Container of the Map
zoom (int) = 10 / Zoom Level of the Map
show_marker (bool) = true / Should a marker be displayed?
title (str) = null / Title of the Marker
zoom_control (bool) = true / Enable or disable Zoom-Conrol
map_type_control (bool) = true / Enable or disable Map-Type-Conrol
scale_control (bool) = true / Enable or disable Scale Control
street_view_control (bool) = true / Enable or disable Street View
overview_map_control (bool) = true / Enable or disable Map Control
map_type (str) = road / Set Map Typ to Road („road“) or satellite („sattelite“)
